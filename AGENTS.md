# AGENTS.md

## 作業方針

このプロジェクトは「仕入原価管理システム」を作成するためのものです。

目的は、店舗仕入れ・ネット仕入れの商品について、仕入日・店舗・商品名・金額・画像・レシートを記録し、売れた時に正確な仕入原価を確認できる状態を作ることです。

## 参照ファイル

最重要仕様書：

- docs/development_spec.md

必ずこの仕様書を読んでから作業してください。

## 進め方

- まず第1段階を完成させる
- 最初からAI画像解析や売上連携まで作り込まない
- Googleフォーム、Googleスプレッドシート、Google Apps Scriptを中心に構築する
- スマホで簡単に入力できることを最優先する
- 削除処理や既存データ破壊を伴う作業以外は、確認待ちせず進める
- 不明点があっても、合理的に仮決めして進める
- 作業後は変更内容、使い方、次にやるべきことをまとめる

## 優先順位

1. Googleフォームから仕入登録できる
2. スプレッドシートに仕入DBとして整理される
3. 未確認リストを作る
4. 日別・店舗別・支払方法別の集計を作る
5. レシートOCRやAI画像解析は後回し

## 禁止事項

- いきなり大規模なAI連携から始めない
- 商品名をレシートだけで完全自動特定しようとしない
- 既存データを勝手に削除しない
- 仕様書と異なる方向に進めない
