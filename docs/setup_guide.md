# 第1段階セットアップ手順

この手順で、Googleフォーム、Googleスプレッドシート、Google Apps Script を使った仕入登録の最小構成を作成できます。

## 作成されるもの

- Googleフォーム: 仕入登録フォーム
- Googleスプレッドシート
  - 仕入DB
  - 未確認リスト
  - 日別集計
  - 店舗別集計
  - 支払方法別集計
  - 設定
- Apps Script
  - フォーム回答の整形
  - 仕入IDの自動採番
  - 画像URLの保存
  - ステータス「未確認」の付与

## 初回セットアップ

1. Google Drive で新しい Apps Script プロジェクトを作成します。
2. `apps-script/Code.gs` の内容を Apps Script の `Code.gs` に貼り付けます。
3. Apps Script のプロジェクト設定で `appsscript.json` を表示できるようにします。
4. `apps-script/appsscript.json` の内容を Apps Script 側の `appsscript.json` に貼り付けます。
5. 関数一覧から `setupPurchaseCostSystem` を選び、実行します。
6. 初回だけ権限確認が出るので許可します。
7. 実行ログに表示される `フォーム編集URL` を開きます。
8. フォームの `商品写真` と `レシート写真` をファイルアップロード項目に変更します。
9. 実行ログに表示される `フォーム入力URL` をスマホで開き、ブックマークします。
10. 作成されたスプレッドシートの `設定` シートにもフォームURLと仕入DB URLが保存されます。

## 写真項目の手動設定

Apps Script の標準フォーム作成機能では、環境によってファイルアップロード項目を自動作成できません。

その場合、フォーム自体は作成されますが、`商品写真` と `レシート写真` は一時的にURL入力欄として作られます。実運用前にフォーム編集画面で以下の作業をしてください。

1. `商品写真` の質問を削除します。
2. 新しい質問を追加します。
3. 質問タイトルを `商品写真` にします。
4. 質問タイプを `ファイルのアップロード` にします。
5. 必須にします。
6. 同じ手順で `レシート写真` も作成します。

タイトルは必ず `商品写真`、`レシート写真` のままにしてください。Apps Script がこの名前でフォーム回答を読み取ります。

## フォーム入力ルール

必須項目は、毎日続けることを優先して最小限にしています。

- 店舗名
- 合計金額
- 商品名メモ
- 商品写真
- レシート写真

任意項目は後から修正できます。

- 仕入日: 空欄なら登録日を仕入日として扱います。
- 支払方法
- 数量: 空欄なら `1` で登録します。
- 型番
- JAN
- ASIN
- 備考

## 仕入IDの形式

仕入IDは以下の形式で自動発行されます。

```text
P-YYYYMMDD-連番
```

例:

```text
P-20260614-001
```

連番は仕入日ごとに採番されます。

## シートの見方

### 仕入DB

フォーム回答を整形したメインDBです。フォーム登録直後は `ステータス` が `未確認` になります。

### 未確認リスト

以下のいずれかに当てはまる行を抽出します。

- ステータスが `未確認`
- 商品名_確定 が空欄
- 数量 が空欄
- 単価 が空欄
- 商品写真URL が空欄
- レシート写真URL が空欄

### 日別集計

仕入日ごとの件数と合計金額を確認できます。

### 店舗別集計

店舗ごとの件数と合計金額を確認できます。

### 支払方法別集計

支払方法ごとの件数と合計金額を確認できます。

## 毎日の使い方

1. 仕入れた直後にスマホでフォームを開きます。
2. 店舗名、合計金額、商品名メモを入力します。
3. 商品写真とレシート写真を添付します。
4. 登録後、その日の最後に `未確認リスト` を見ます。
5. 必要に応じて `商品名_確定`、`単価`、`数量`、`支払方法` などを補完します。
6. 確認が終わった行は `ステータス` を `確認済` に変更します。

## 注意点

- 商品名はレシートだけで特定しようとしないでください。
- 商品写真と商品名メモを主情報として残してください。
- レシートOCR、AI画像解析、売上連携は第2段階以降で追加します。
- 既存の `仕入DB` の行は削除しない運用を前提にしています。
